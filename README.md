
# Simple Auth App
Mini project for crud todo list
## Run Locally

Clone the project

```bash
  git clone https://github.com/azkiya/todo-app
```

Go to the project directory

```bash
  cd todo-app
```

Install dependencies

```bash
  composer install
```
```bash
  npm install
```

Setting environment

```bash
  create file .env minimal require from env.example, 
```

Start the server

```bash
  php artisan serve
```

Run automation test

```bash
  php artisan test
```


## Tech Stack

**Server:** PHP

**Framework** Laravel 11
